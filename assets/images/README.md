# Tripoli Images

Placeholder images are included. For better quality, download Tripoli photos:

1. Go to https://commons.wikimedia.org/wiki/Category:Tripoli,_Lebanon
2. Download and save as:
   - intro_1.jpg - Whole Tripoli (city view)
   - intro_2.jpg - قلعة التل (Citadel of Tripoli / Qala'at Sanjil)
   - intro_3.jpg - Old city/souks
   - intro_4.jpg - Mosque
   - downtown.jpg - Street/market

Or run: `dart run scripts/download_images.dart` (requires internet)
